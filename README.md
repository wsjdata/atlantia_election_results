| **Column Name**                                              | **Description**                                                                                                 |
|--------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------|
| `province` | The larger administrative region or area where the election is taking place (like a state or province).         |
| `district` | A smaller administrative division within the province that oversees the election in that area (like a county or district). |
| `precinct` | The smallest administrative unit where voters go to cast their ballots (like a polling station or local precinct). |
| `number_of_voters_included_in_the_list` | Total number of registered voters in the precinct who are eligible to vote.                                     |
| `number_of_ballots_received_by_the_precinct_commission` | Total number of ballots that the precinct received to distribute to voters.                                     |
| `number_of_ballots_issued_to_early_voters` | Number of ballots given to voters who voted before the official election day.                                   |
| `number_of_ballots_issued_at_the_polling_station` | Number of ballots given to voters who voted at the polling station on election day.                             |
| `number_of_ballots_issued_outside_the_polling_station` | Number of ballots given to voters outside the polling station, usually for those who cannot come to the polling station. |
| `number_of_cancelled_ballots` | Number of ballots that were cancelled or invalidated and not counted.                                           |
| `number_of_ballots_in_portable_ballot_boxes` | Number of ballots collected in portable boxes, typically used for mobile voting services.                      |
| `number_of_ballots_in_stationary_ballot_boxes` | Number of ballots collected in the stationary boxes at the polling station.                                     |
| `number_of_invalid_ballots` | Number of ballots that were marked incorrectly or were otherwise invalid and not counted.                       |
| `number_of_valid_ballots` | Number of ballots that were marked correctly and counted.                                                       |
| `number_of_absentee_certificates_received_by_the_precinct_commission` | Number of certificates allowing absentee voting received by the precinct.                                   |
| `number_of_absentee_certificates_issued_at_the_polling_station` | Number of absentee voting certificates given out at the polling station.                                      |
| `number_of_voters_who_voted_with_absentee_certificates_at_the_polling_station` | Number of people who voted at the polling station using absentee certificates.                           |
| `number_of_cancelled_unused_absentee_certificates` | Number of unused absentee certificates that were cancelled.                                                     |
| `number_of_absentee_certificates_issued_by_the_territorial_election_commission` | Number of absentee voting certificates issued by a higher election authority.                                |
| `number_of_lost_absentee_certificates` | Number of absentee voting certificates that were lost.                                                          |
| `number_of_lost_ballots` | Number of ballots that were lost during the election process.                                                   |
| `number_of_ballots_not_accounted_for_upon_receipt` | Number of ballots that were missing or not properly recorded when received by the precinct.                     |
